## 서비스 소개

데이터 통합 플랫폼 기반 지능형 IoT 관리 솔루션의 **설계 및 제안 자료**를 웹 애플리케이션으로 제공합니다.  
분산된 IoT 데이터 통합, 실시간 모니터링, 원격 제어, AI 기반 분석을 아우르는 제안 내용을 **3가지 웹 화면**으로 볼 수 있습니다.

**통합 관리 범위 (요약)**  
프로토콜: TCP, MQTT, REST API | 형식: Hex Binary, JSON, CSV | 원천: IoT 센서, 파일 배치, RDBMS, NoSQL

**주요 특징**

- **3가지 화면**: 프레젠테이션(개요) → 대시보드(상세) → 설계 문서(문서)
- 실시간 차트·다이어그램, 반응형 레이아웃
- URL 해시로 화면·슬라이드·문서 직접 링크 공유 가능

## 접속 방법

- **접속 URL**: 서비스 제공자가 안내한 주소로 접속 (예: `https://도메인/` 또는 `http://localhost:3000`)
- **화면 전환**: 우측 상단 **프레젠테이션** | **대시보드** | **설계 문서** 버튼으로 전환
- **직접 링크**: `#presentation/...`, `#dashboard/...`, `#docs/...` 해시를 URL에 붙여 특정 화면·슬라이드·문서로 바로 이동

---

## 웹 화면 (3가지)

접속 시 **프레젠테이션**이 기본 화면이며, 우측 상단에서 **대시보드**, **설계 문서**로 전환할 수 있습니다.

### 1. 프레젠테이션 (첫 화면)

- **내용**: 제안 개요용 슬라이드 7장 (제목 → 데이터 문제 정의 → 표준화·통합 → 플랫폼/아키텍처 → 모니터링·제어·분석 → 로드맵 → ROI·비즈니스 임팩트)
- **이용**: 키보드/마우스로 이동, 슬라이드 인디케이터·진행률, 전체화면 지원
- **URL**: `#presentation/{slide-id}` (예: `#presentation/title`)
- **대상**: 고객사, 경영진, 의사결정자

### 2. 대시보드

- **내용**: 7개 프레임 (데이터 문제 정의, 표준화, 통합 플랫폼, 실시간 모니터링, 원격 제어·OTA, 지능형 분석, 데이터 활용 확장). 차트·KPI 카드·상세 기술 정보
- **이용**: 프레임별 이동, 실시간 시각화
- **URL**: `#dashboard/{frame-id}` (예: `#dashboard/problem`)
- **대상**: 기술팀, 아키텍트, 개발팀

### 3. 설계 문서

- **내용**: 5개 문서 (설계 문서 가이드, 서비스 개요, 프로세스 플로우, 기술 분석, 웹 애플리케이션 가이드). Mermaid 다이어그램(확대·전체 보기), 섹션 네비게이션, 반응형
- **이용**: 문서 선택 → 섹션 클릭으로 이동, 다이어그램 확대·드래그
- **URL**: `#docs/{doc-id}/{section-id}` (예: `#docs/process-flow`)
- **대상**: 모든 이해관계자, 상세 검토가 필요한 사용자

## 화면별 보기 내용 요약

**프레젠테이션 슬라이드**  
1. 문제점 분석 2. 솔루션 개요 3. 시스템 아키텍처 4. 핵심 기능·이점 5. 구현 로드맵 6. ROI 분석

**대시보드 프레임**  
1. 데이터 문제 정의 2. 데이터 표준화 3. 데이터 통합 플랫폼 4. 실시간 모니터링 5. 원격 제어·OTA 6. 지능형 분석 7. 데이터 활용 확장
